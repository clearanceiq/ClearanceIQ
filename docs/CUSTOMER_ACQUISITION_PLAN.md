# ClearanceIQ — Customer Acquisition Plan

**Goal:** Move from zero real users to paying import professionals using the tools daily.

**Status today:** All tools coded, CORS fix in CDN queue, no live traffic yet.

---

## Phase 1 — Seed the First 100 Users (Week 1–2)
**Focus: prove that real importers actually use the tools**

**Channels:**
1. **Reddit (highest ROI, zero cost)**
   - r/importing — post as "free HTS lookup + duty estimator" (moderate rules, value-first)
   - r/FulfillmentByAmazon — "free tool to estimate landed costs before shipping from China"
   - r/ecommerce — general post about avoiding surprise customs fees
   - r/supplychain — professional angle for freight forwarders
   - Format: short pain-point hook + direct link + no affiliate spam
   - Target: 5 quality posts per day, 1 comment on 20 existing threads

2. **LinkedIn (freight forwarders and importers)**
   - Open to public, not invite-only (except ClearanceIQ itself)
   - Search: "customs broker", "import manager", "supply chain director"
   - Send connection request + 1-sentence value message
   - Don't spam; engage on their posts first
   - Target: 30 connections/day, 5 conversations

3. **Facebook Groups**
   - Amazon FBA Importing from China
   - US Importers & Wholesalers
   - Freight Forwarder Network
   - Format: same pain-point hook, link in comments, not top-level ad
   - Target: 2–3 groups, 1 post each per day

4. **Cold Email (if you have lists)**
   - If no list, skip. Reddit/LinkedIn are faster.
   - If you build a list later, use sequence: tool → case study → paid plan

**Expected seed conversion:**
- 500 visits from Reddit/LinkedIn/FB
- 50 tool users
- 10 email captures
- 5 daily active users

## Phase 2 — Content Engine (Week 3–6)
**Focus: organic traffic that compounds**

**SEO Blog (highest priority, per loop):**
- Target long-tail keywords importers actually search:
  1. "How to find HTS code for [product]"
  2. "Duty rate calculator for imports from [country]"
  3. "What is a CBP hold and how to avoid it"
  4. "Continuous vs single transaction bond"
  5. "Section 301 tariffs explained"
  6. "Landed cost formula for Amazon FBA"
  7. "Customs broker vs self-filing"
  8. "ISF filing requirements"
  9. "First time importer guide USA"
  10. "How to read a commercial invoice for customs"
- Publish 2–3 articles per week (500–800 words, practical, no fluff)
- Each article embeds the relevant tool inline (HTS tool in HTS post, etc.)
- Each article ends with "Save this as PDF / email me the guide" → lead capture

**YouTube Shorts (second priority):**
- 60-second scripts showing "3 things CBP checks before release"
- "One tool to calculate duty before you ship"
- "Why your shipment got held (and how to fix it)"
- Post daily at 9AM EST
- Link to ClearanceIQ in bio

**Newsletter (month 2+):**
- Weekly "Customs Tip of the Week"
- Curated HTS updates, tariff changes, CBP news
- Convert free readers to paid

## Phase 3 — Partnerships & Distribution (Month 2+)
**Focus: piggyback on existing networks**

- **Freight forwarder referral program**
  - Give them a branded "Landed Cost Calculator" with their logo
  - They share it with their shippers
  - Commission on converted paid users

- **Customs broker co-branding**
  - Offer free limited API access
  - They integrate into their client portal

- **E-commerce tool stack**
  - Integrate with Shopify apps or Amazon seller tools
  - "Add ClearanceIQ to your import workflow"
