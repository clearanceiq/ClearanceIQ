# ClearanceIQ — Product Spec: Free-Safety / Paid-Power + Trust

*Derived from exec roundtable (Musk, Bezos, Gates, Ma). Status: draft for review.*

---

## 1. Core Model (consensus)

**Free = Safety. Paid = Power.**

- Duty/HTS check, CBP hold decoder, compliance & supplier checklists stay **free forever**.
- No paywall on anything that prevents a bad import decision (fear-tax banned — Ma/Bezos rule).
- Paid tier unlocks **scale, memory, and automation** — not the basics.

---

## 2. Free Tier (anonymous + signed)

| Capability | Anonymous (5/day) | Signed-free (100/day) |
|------------|-------------------|------------------------|
| HTS lookup | ✅ | ✅ |
| Duty calculator | ✅ | ✅ |
| Bond estimator | ✅ | ✅ |
| CBP hold decoder | ✅ | ✅ |
| Compliance / supplier checklists | ✅ | ✅ |
| Saved history | ❌ | ✅ (last 30 days) |
| Email alerts on limit reset | ❌ | ✅ |

**Signup promise:** "100 free lookups/day, your history saved, no card."

---

## 3. Paid Tier ("Pro" — must actually exist)

| Capability | Pro |
|------------|-----|
| Unlimited daily lookups | ✅ |
| Permanent saved history + CSV export | ✅ |
| Bulk upload (spreadsheet → duty for 500 SKUs) | ✅ |
| Section 301 exclusion checker by HTS | ✅ |
| FBA landed-cost calculator (freight + 3PL + duty) | ✅ |
| API access (own scripts / Shopify / ERP) | ✅ |
| Team seats (forwarder shared account) | ✅ |
| White-glove: ruling reference links | ✅ |

**Pricing:** single transparent tier, monthly. No dead "learn more" link — button goes to a real checkout.

---

## 4. Flywheel (Bezos/Ma compromise)

1. Free tool solves a real fear → user trusts us
2. Signup → history saved → reason to return daily
3. Daily return → hits limit → *wants* Pro (not pushed to it)
4. Happy SMB tells peers → bottom-up growth (Ma)
5. Forwarder sees demand → licenses API for clients (Gates leverage)

**Instrument:** track signup→return→Pro-conversion. That ratio is the real metric, not lookup count.

---

## 5. Trust Page (closes Gates/Ma objection)

Must ship before any paid push:

- **About** — who runs ClearanceIQ, why it exists
- **Credentials** — broker affiliation / advisory, disclaimers clearly stated
- **Methodology** — where duty data comes from, how often updated, what "indicative" means
- **Data promise** — what we store, what we don't, delete-on-request
- **Contact** — real channel, not a black hole

No claims we can't back. "Planning reference only" stays on every result.

---

## 6. Dataset as Asset (Musk/Gates)

Every lookup, correction, and ruling = proprietary training data.

- Log: input, result, user correction, outcome
- Use to improve matching + eventually power features competitors can't copy
- Serve improved insights back to free users (Ma's "get smarter each time")

## 8. Non-negotiables (all four)

- No paywall on safety tools
- No dead links
- No fake credentials
- Dataset captured deliberately from day one
