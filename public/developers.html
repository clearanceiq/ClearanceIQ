Developer Quickstart

HTS lookups and duty estimates, two requests away.

Sign up for a free API key, then call /api/v1/hts in minutes.

Quickstart

  1. Create your free API key.
  2. Call the API.

1. Create a key

curl -s -X POST https://clearance-iq.com/api/auth \
  -H "content-type: application/json" \
  -d '{"email": "you@company.com"}'

Returns key, tier, and usage instructions.

2. Call the HTS endpoint

curl -s "https://clearance-iq.com/api/v1/hts?q=smartphone" \
  -H "x-api-key: ciq_XXXXXXXXXXXXXXXXXXXXXXXX"

3. Example response

{
  "ok": true,
  "code": "8517.12.0095",
  "description": "Smartphones and base stations",
  "dutyRate": 0,
  "adcvd": "low",
  "note": "This category is sometimes flagged for AD/CVD checks. Confirm before booking."
}

Endpoints

/api/v1/hts

GET /api/v1/hts?q=<query>
X-API-Key: ciq_... (optional)

Search HTS codes and descriptions. Returns a single best match with duty rate and AD/CVD flag.

/api/v1/duty-calc

GET /api/v1/duty-calc?...

Duty estimate with landed cost inputs. Works returns JSON with estimates based on product and cost basis.

/api/v1/bond

GET /api/v1/bond?...

Bond estimator input flow. Accepts shipment value and bond type inputs.

Error Codes

HTTP Code Meaning
400 q required Missing search query
400 invalid_query Malformed request body or query
400 no_match No HTS match found
401 unauthorized Missing or invalid API key (if required)
404 not_found Unknown endpoint
429 rate_limit Daily limit reached

Bodies use application/json. Preflight OPTIONS returns 204.
