# BotFather Setup Commands for KasiSetelbot
# Copy-paste these into @BotFather in order

## Step 1: Create bot (skip if already done)
/newbot

## Step 2: Set bot name
# BotFather will ask for bot username. Type: KasiSetelbot

## Step 3: Set bot description
/setdescription
# Select: KasiSetelbot
# Paste:
🤖 KasiSetelbot — privacy-first business partner.
Kelola dokumen, chat pelanggan, dan jadwal otomatis dengan AI. Flat RM59/bulan.
Coba 7 hari gratis di @KasiSetelbot.
🔒 Data tidak pernah dibagikan, dijual, atau dipakai untuk model apa pun.

## Step 4: Set bot about text
/setabouttext
# Select: KasiSetelbot
# Paste:
AI assistant for Malaysian & Indonesian businesses. Bahasa Malaysia + English. Privacy-first. Flat fee.

## Step 5: Set bot commands (for group/chat hint)
/setcommands
# Select: KasiSetelbot
# Paste:
start - Show welcome message and current tier
help - Help and contact info
persona - Choose AI persona (Operator, Writer, Translator, Researcher, Customs)
upgrade - Upgrade plan (Starter / Standard / Private)

## Step 6: (Optional) Enable Payments
/mybots
# Select KasiSetelbot -> Bot Settings -> Payments
# Choose your payment provider (Stripe recommended for MY)
# Note: Telegram Stars requires separate setup via @BotFather -> /mybots -> Payments

## Step 7: (Optional) Set group privacy
# If you want the bot to read all messages in groups:
# /setjoingroups -> Turn OFF privacy mode
